🎯 1 Introdução e Objetivo: </strong>

Este projeto consiste no desenvolvimento de uma Landing Page de Portfólio Pessoal, criada como parte da atividade prática extra do curso de Introdução à Programação Front-End e Back-End do SCTEC - SENAI/SC LAB365. 

O objetivo principal é consolidar os conhecimentos adquiridos durante as aulas, apresentando de forma estruturada minha trajetória formativa, habilidades técnicas e produções na área de tecnologia.

Esta página foi planejada para servir como um instrumento inicial de apresentação profissional, sendo um ambiente digital centralizado para demonstrar projetos e competências a recrutadores e parceiros do ecossistema de tecnologia de Santa Catarina.

🛠️ <strong>2 Tecnologias Utilizadas:</strong>

O projeto utiliza o trio fundamental do desenvolvimento Web:

<strong>[x] HTML5:</strong> Utilizado para a estruturação semântica de todo o conteúdo da página, garantindo acessibilidade e organização lógica das seções.

<strong>[x] CSS3:</strong> Responsável pela estilização visual, definição de layout, tipografia, cores e garantir que a página seja visualmente atraente e organizada.

<strong>[x] JavaScript:</strong> Implementado para adicionar interatividade à página, cumprindo a exigência de uso das três linguagens.

🗂️ <strong>3 Estrutura do Projeto: </strong>

A landing page está dividida em seções para uma apresentação completa:

<strong>[x] Cabeçalho/Identificação:</strong> Contém o nome do estudante e uma breve apresentação.

<strong>[x] Sobre Mim (Descrição Pessoal):</strong> Um texto detalhado sobre minha formação e interesses na área de programação.

<strong>[x] Habilidades Técnicas:</strong> Listagem de competências técnicas. 

<strong>[x] Soft Skills:</strong> Listagem de competências comportamentais.	

<strong>[x] Portfólio de Projetos:</strong> Uma vitrine contendo projetos, detalhando o nome, descrição, tecnologias envolvidas e links de demonstração.

🚀 <strong>4 Instruções de Execução:</strong>

Para visualizar o projeto localmente, siga os passos abaixo:

Extraia o conteúdo do arquivo compactado (.zip ou .rar).

Certifique-se de que os arquivos index.html, style.css e script.js estão na mesma pasta.

Abra o arquivo index.html em qualquer navegador web moderno (Chrome, Firefox ou Edge).
