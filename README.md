Documentação do Projeto: Portfólio Profissional - Angélica Daiane Lorenz - SCTEC SENAI LAB365
1. Introdução e Objetivo

Este projeto consiste no desenvolvimento de uma Landing Page de Portfólio Pessoal, criada como parte da atividade prática extra do curso de Introdução à Programação Front-End e Back-End do SCTEC - SENAI/SC LAB365. O objetivo principal é consolidar os conhecimentos adquiridos durante as aulas, apresentando de forma estruturada minha trajetória formativa, habilidades técnicas e produções na área de tecnologia.

Esta página foi planejada para servir como um instrumento inicial de apresentação profissional, sendo um ambiente digital centralizado para demonstrar projetos e competências a recrutadores e parceiros do ecossistema de tecnologia de Santa Catarina.
2. Tecnologias Utilizadas

Para cumprir os requisitos técnicos de avaliação e garantir o score máximo em "Linguagens e Arquivos", o projeto utiliza o trio fundamental do desenvolvimento Web:

    HTML5: Utilizado para a estruturação semântica de todo o conteúdo da página, garantindo acessibilidade e organização lógica das seções.

    CSS3: Responsável pela estilização visual, definição de layout, tipografia, cores e garantir que a página seja visualmente atraente e organizada.

    JavaScript: Implementado para adicionar interatividade à página, cumprindo a exigência de uso das três linguagens.

3. Estrutura do Projeto

Conforme solicitado no edital, a landing page está dividida em seções obrigatórias para uma apresentação completa:

    Cabeçalho/Identificação: Contém o nome do estudante e uma breve apresentação.

    Sobre Mim (Descrição Pessoal): Um texto detalhado sobre minha formação e interesses na área de programação.

    Habilidades (Skills): Listagem de competências técnicas (Hard Skills) e comportamentais (Soft Skills).

    Portfólio de Projetos: Uma vitrine contendo projetos desenvolvidos, detalhando o nome, descrição, tecnologias envolvidas e links de demonstração.

4. Instruções de Execução

Para visualizar o projeto localmente, siga os passos abaixo:

    Extraia o conteúdo do arquivo compactado (.zip ou .rar).

    Certifique-se de que os arquivos index.html, style.css e script.js estão na mesma pasta.

    Abra o arquivo index.html em qualquer navegador web moderno (Chrome, Firefox ou Edge).

5. Reflexão Técnica

O desenvolvimento deste desafio permitiu aplicar na prática conceitos de design responsivo e manipulação de scripts. A preocupação com a organização do código-fonte e a correta documentação visa não apenas o atendimento aos critérios de avaliação, mas também a adoção de boas práticas de mercado, como o versionamento e a clareza técnica.
