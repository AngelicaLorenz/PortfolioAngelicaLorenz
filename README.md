🎯 1 Introdução e Objetivo:

Este projeto consiste no desenvolvimento de uma Landing Page de Portfólio Pessoal, criada como parte da atividade prática extra do curso de Introdução à Programação Front-End e Back-End do SCTEC - SENAI/SC LAB365. 

O objetivo principal é consolidar os conhecimentos adquiridos durante as aulas, apresentando de forma estruturada minha trajetória formativa, habilidades técnicas e produções na área de tecnologia.

Esta página foi planejada para servir como um instrumento inicial de apresentação profissional, sendo um ambiente digital centralizado para demonstrar projetos e competências a recrutadores e parceiros do ecossistema de tecnologia de Santa Catarina.

🛠️ 2 Tecnologias Utilizadas:

O projeto utiliza o trio fundamental do desenvolvimento Web:

HTML5: Utilizado para a estruturação semântica de todo o conteúdo da página, garantindo acessibilidade e organização lógica das seções.

CSS3: Responsável pela estilização visual, definição de layout, tipografia, cores e garantir que a página seja visualmente atraente e organizada.

JavaScript: Implementado para adicionar interatividade à página, cumprindo a exigência de uso das três linguagens.

🗂️ 3 Estrutura do Projeto:

A landing page está dividida em seções para uma apresentação completa:

[x] Cabeçalho/Identificação: Contém o nome do estudante e uma breve apresentação.

[x] Sobre Mim (Descrição Pessoal): Um texto detalhado sobre minha formação e interesses na área de programação.

[x] Habilidades Técnicas: Listagem de competências técnicas. 

[x] Soft Skills: Listagem de competências comportamentais.	

[x] Portfólio de Projetos: Uma vitrine contendo projetos, detalhando o nome, descrição, tecnologias envolvidas e links de demonstração.

🚀 4 Instruções de Execução:

Para visualizar o projeto localmente, siga os passos abaixo:

Extraia o conteúdo do arquivo compactado (.zip ou .rar).

Certifique-se de que os arquivos index.html, style.css e script.js estão na mesma pasta.

Abra o arquivo index.html em qualquer navegador web moderno (Chrome, Firefox ou Edge).
